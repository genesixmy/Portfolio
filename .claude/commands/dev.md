# Start Development Server

Run the Next.js development server on port 3000.

This command starts the development server with hot reload enabled. The app will be available at http://localhost:3000

**Usage:**
```bash
npm run dev
```

**What happens:**
- Starts Next.js dev server
- Enables hot module reloading
- Watch for file changes in src/ and pages/
- Compiles TypeScript on the fly
